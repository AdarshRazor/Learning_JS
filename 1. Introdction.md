## Escaping literal quotes in Strings

```js
var myStr = "Hi, my name is \"Adarsh Anand\"";
//use "\" to escape the quote
console.log(myStr);
var myStr = 'Hi, my name is "Adarsh Anand"';
// or use ' to escape the double quote
```

## Escape sequence 

```js
/***
CODE OUTPUT
\' - single quote
\" - double quote
\n - new line
\t - tab
\\ - backslash
\' - single quote
\" - double quote
\b - backspace
\f - form feed
\r - carriage return
\v - vertical tab
***/

var myStr = "FirstLine\n\t\\SecondLine\nThirdLine";
console.log(myStr);
```
